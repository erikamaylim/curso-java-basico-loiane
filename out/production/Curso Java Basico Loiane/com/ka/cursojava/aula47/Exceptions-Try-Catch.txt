Tratamento de exceções no Java
    ⋄ Palavras-chave deste módulo:
        • Try, catch, throw, throws e finally

    ⋄ Para esta aula:
        • Try e catch

        try {
            // bloco que é monitorado para erros
        }
        catch (TipoDaException exception){
            // Tratamento do erro
        }

    ⋄ Não tratar erros faz a execução do programa terminar

    ⋄Tratar erros:
        • Permite que o programa execute uma determinada lógica caso o erro ocorra
        • Permite que p programa continue a execução
